# A statistical genetics framework for discovering druggable protein complexes in human plasma

This repository contains all supplementary tables for the study: **"A statistical genetics framework for discovering druggable protein complexes in human plasma."**


## Supplementary Tables

### Table 1: Description of 2,826 proteins from the UK Biobank Pharma Proteomics Project
This table summarizes annotations and summary association statistics for 2,826 plasma proteins profiled in the UK Biobank Pharma Proteomics Project (UKB-PPP), used in *cis*-pQTL analyses.

### Table 2: 262 significant HDL-L results with experimental evidence in STRING
Contains results for protein pairs showing significant genetic correlations that are further supported by experimental evidence within the STRING database.

### Table 3: Top 50 gene pairs ranked by co-occurring curated pathways
Ranked list of gene pairs based on the number of shared curated pathways within the **MSigDB C2:CP** (Curated Pathways) collection.

### Table 4: High-confidence, complex-specific druggable pockets
Druggable pockets predicted by **DoGSite-Scorer**, specifically filtered for high-confidence scores and relevance to the identified protein complexes.

These tables are provided in **.csv** format to support the findings presented in the main manuscript.
